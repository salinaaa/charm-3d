C code: there is no separate boolean datatype. We use int, as recommended by the standard, with 0 meaning false and everything else meaning true. Thus, the only safe way of testing for truth is if (var) (or, if you insist, if (var != 0)). You may technically use if (var == 0) to test for falseness but we recommend if (!var). The NULL pointer always evaluates to false, but we try to use the explicit test if (pointer == NULL) throughout.
